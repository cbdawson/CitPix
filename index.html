    Collaboratively-developed open source, free app to post photo & standardized metadata to social media platforms as part of citizen-science projects

    Project maintained by cbdawson Hosted on GitHub Pages — Theme by mattgraham

    Overview

    Many citizen science projects collect volunteer photos via social media. However, for projects that require exact locations and time/date, consistent and accurate reporting can be a problem. For example, some social media platforms such as Twitter, strip image exif data, requiring volunteers to compile and add that information manually.

    This San Francisco Science Hack Day proposal is to develop a simple, free application citizen science projects can have participants use to post photos to social media channels. The app would automatically extract image exif data and post it in a consistent, standardized approach.

    This app could be used by King Tides Project volunteers as well as any other interested citizen science projects.
